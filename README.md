# TraceWear Releases

Public downloads, release notes, policies, checksums, and support templates for
TraceWear by Korvex. The application source remains in a separate private
repository and is not published here.

- Prepared site: https://renoman1195.github.io/TraceWear-Releases/ (deployment paused)
- Support: https://github.com/Renoman1195/TraceWear-Releases/issues
- Current release: v1.0.0

The Windows installer starts a 14-day Pro evaluation and then continues in
Basic mode without deleting local work. The Pages site remains prepared for a
future deployment or migration to the publisher's primary website.

Do not post sensitive vehicle, maintenance, personnel, organization,
classified, controlled, or export-restricted information in public issues.
