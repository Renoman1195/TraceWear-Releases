Release notes

TraceWear 1.0.0

Initial professional-prototype release for Windows x64, published by Jacob Brown.

Included

  • Offline simulation, recorded replay, capture import, Korvex analysis, Help, and report export.
  • Local Markdown and JSON maintenance reports and privacy-safe local bug-report export.
  • Signed update channel with non-blocking checks, user confirmation, progress, and passive Windows installation.
  • Global interface recovery handling and actionable failure states.
  • PCAN, Kvaser, and RP1210 preview paths with safe missing-driver handling.

Important limits

  • Bundled parts, costs, captures, and predictions are prototype/reference data.
  • Vendor drivers are not bundled.
  • Real-adapter field validation and authoritative military maintenance data are outside the 1.0 claim.
  • TraceWear is not an operational maintenance authority.
