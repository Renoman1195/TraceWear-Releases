Release notes

TraceWear 1.0.0

Initial professional-prototype release for Windows x64, published by Jacob Brown.

Access model

  • One installer starts a 14-day Pro evaluation on first launch.
  • After the evaluation, TraceWear continues in Basic mode without deleting local work.
  • Basic retains simulation, bundled replay, one bundled profile, core risk results, Help, updates, and local bug-report export.

Included

  • Offline Basic simulation, bundled replay, Korvex core risk analysis, and Help.
  • Pro evaluation access to capture import, advanced cost and mission analysis, report export, synchronization, additional profiles, and preview live hardware.
  • Signed update channel with non-blocking checks, user confirmation, progress, and passive Windows installation.
  • Global interface recovery handling and actionable failure states.
  • PCAN, Kvaser, and RP1210 preview paths with safe missing-driver handling.

Important limits

  • The initial trial entitlement is local-device enforcement; paid Pro sales require a server-issued entitlement.
  • Bundled parts, costs, captures, and predictions are prototype/reference data.
  • Vendor drivers are not bundled.
  • Real-adapter field validation and authoritative military maintenance data are outside the 1.0 claim.
  • TraceWear is not an operational maintenance authority.
